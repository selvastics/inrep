Interactive Response Theory Package for R

About inrep

inrep is a comprehensive R package for implementing interactive response theory (IRT) assessments and computerized adaptive testing (CAT). It provides robust tools for creating, administering, and analyzing psychological and educational assessments.

Installation & Usage

To use this package, you need to install it in an R environment: 

# Install from source
R CMD INSTALL inrep_1.0.0.tar.gz

# Or use devtools
devtools::install_local("inrep_1.0.0.tar.gz")

# Load the package
library(inrep)

# Launch the study
launch_study()

Why This Won't Work on Vercel

Vercel is a platform designed for deploying web applications built with:

  • React, Next.js, Vue.js
  • Node.js applications
  • Static websites
  • Serverless functions

R packages are statistical computing tools that run in R environments, not web applications. They cannot be deployed on Vercel.

Alternative Deployment Options

If you want to deploy this as a web application, consider:

  • Shiny Server - For hosting Shiny applications
  • RStudio Connect - For enterprise R deployment
  • Docker containers - For containerized R applications
  • Cloud platforms - AWS, Google Cloud, Azure with R support
